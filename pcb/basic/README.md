# Basic PCB

OSHWLab page: https://oshwlab.com/ganzevich/PowMr-Hybrid-Inverter

## PSU module connection
Using standard XL4005 module:

| Main board | XL4005 PSU        |
|------------|-------------------|
| GND pad    | PSU  ` - ` input  |
| 12V pad    | PSU `+` input     |
| 5V pad     | PSU  `+`  output  |
